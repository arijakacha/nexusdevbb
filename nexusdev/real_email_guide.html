📊 Email Delivery Options (Symfony Mailer)

🔵 Option 1: Gmail (Current)

Pros: Free, easy setup, reliable

Cons: 500 emails/day limit, development only

MAILER_DSN=gmail://your-email@gmail.com:app-password@default

🟡 Option 2: SendGrid (Recommended for Production)

Pros: 100 emails/day free, professional, good deliverability

Cons: Requires API key setup

# Install: composer require symfony/sendgrid-mailer MAILER_DSN=sendgrid://YOUR-API-KEY@default

🟢 Option 3: Mailgun (Professional)

Pros: 5,000 emails/month free, excellent deliverability

Cons: More complex setup

# Install: composer require symfony/mailgun-mailer MAILER_DSN=mailgun://YOUR-API-KEY@default

🔴 Option 4: Amazon SES (AWS)

Pros: 62,000 emails/month free, very reliable

Cons: AWS account required, complex setup

# Install: composer require symfony/amazon-mailer MAILER_DSN=ses://ACCESS_KEY:SECRET_KEY@default

🚀 Upgrade to SendGrid (Recommended)

If you want better deliverability than Gmail, upgrade to SendGrid:

Step 1: Sign up for SendGrid
https://sendgrid.com/
Step 2: Get API Key
  • Go to Settings → API Keys
  • Create API Key with "Mail Send" permissions
Step 3: Install Symfony SendGrid Mailer
composer require symfony/sendgrid-mailer
Step 4: Update .env
MAILER_DSN=sendgrid://YOUR-API-KEY@default
